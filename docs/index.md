# community-church-site

Community church website. React + Vite single-page app deployed to GitHub Pages.

| Detail | Value |
|---|---|
| Repo | [github.com/aaronjenkins/community-church-website](https://github.com/aaronjenkins/community-church-website) |
| Local | `~/Code Projects/community-church-site` |
| Deploy | GitHub Pages, auto-deploy on push to main |

## Stack

- **React 19** + **Vite 6**
- GitHub Actions workflow for build and deploy to GitHub Pages
- ESLint for linting

## Commands

```bash
cd ~/Code\ Projects/community-church-site
npm run dev      # local dev server
npm run build    # production build
npm run lint     # eslint
```
